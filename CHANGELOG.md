## [1.0.0] - 2026-04-08

### Added

- Base62 encoding and decoding (`encode`/`decode`) using the standard `0-9A-Za-z` alphabet with Bitcoin/base-x style leading zero preservation
- Support for both Erlang and JavaScript targets
